Dúvidas Frequentes

A partir de 11 anos. Mas crianças menores também podem participar. Nesse caso, recomendamos que os pais participem ativamente, acompanhando as aulas junto com os seus filhos.

Sim, adultos podem e devem participar, tanto se quiserem usar esse aprendizado para si próprios quanto se o objetivo for transmitir esse conhecimento para seus filhos, netos, sobrinhos ou alunos.

Via Zoom, ao final de cada módulo. As aulas ao vivo terão como pauta os conteúdos daquele módulo (revisão e esclarecimento de dúvidas).

✅ Como Usar no WordPress

  1. 📦 Instalar ACF: Plugin "Advanced Custom Fields" ativo
  2. ✏️ Editor Gutenberg: Clique no "+" para adicionar bloco
  3. 🏷️ Categoria "BrightMinds": Procure pelos blocos na categoria
  4. ⚙️ Configurar Campos: Use o painel lateral direito
  5. 📱 Preview em Tempo Real: Veja as mudanças no editor
  6. 🌐 Publicar: Os blocos aparecem no frontend

🚀 Características Técnicas

  • ✅ Modular: Cada bloco em pasta própria
  • ✅ Auto-carregamento: Registro automático via functions.php
  • ✅ Responsivo: Adaptado para desktop e mobile
  • ✅ Acessível: Suporte a leitores de tela e navegação por teclado
  • ✅ SEO-friendly: HTML semântico e otimizado
  • ✅ Performance: Lazy loading para vídeos YouTube
  • ✅ Editável: Interface amigável no admin
  • ✅ Expansível: Fácil adicionar novos blocos
